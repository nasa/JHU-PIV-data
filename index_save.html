PIV Measurements of the Flow Field in the Rotor Tip Region of a Compressor

Researchers at Johns Hopkins University have conducted a set of experiments under a NASA Research Announcement (NRA), that measure and demonstrate the effect of varying rotor tip clearances in a low-speed axial compressor. Detailed stereo PIV measurements of the tip flow field were obtained in their optically index-matched facility. To help understand the complex flow phenomena associated with the tip clearances, the researchers used rotor and stator blades based on the first 1.5 stages of the Low-Speed Axial Compressor (LSAC) at NASA Glenn Research Center. Various casing treatments, semicircular, U-shape and S-shape grooves, were then tested to determine the effect on the large tip clearance flow and their effect to the onset of stall.


For the smooth wall test cases, data was collected at both high flow coefficient, φ = 0.35, and low flow, φ = 0.25, which also represented pre-stall conditions. The tip clearance over the rotor were set (and measured) to be 0.49% (small tip gap) and 2.3% (large tip gap) of the total axial chord.


Test data with the semi-circle casing treatment were obtained for three flow coefficients: φ = 0.25, φ = 0.35 and φ = 0.38 (design condition), and data collected with the U-groove and S-groove casing treatments were at a flow coefficient of φ = 0.25 and φ = 0.38.


The tip clearance for all casing treatment cases were measured to be 1.8 mm


JHU PIV data are stored in MATLAB binary *.mat files. The *.mat file contains several arrays including velocity, vorticity, TKE and turbulence quantities. A description of all arrays can be found here.The zip file contains, nominally, 14 data files for each test condition/configuration. Each data file represents a chordwise location (s/c) of the measurement plane. For more detail about test cases, configurations and measurement planes (as well as copy of array descriptions) you can download the xlsx file, Supplemental_data.xlsx
